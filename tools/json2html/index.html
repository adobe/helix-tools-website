JSON2HTML Configuration

The JSON2HTML Configuration tool provides an interface to configure the JSON2HTML endpoint for an AEM project.

Access to configuration is determined by user permissions.

To configure JSON2HTML, log in to the AEM sidekick for the relevant site in a separate tab, then return here to manage the configuration.
