Content Score

Content Score is a browser tool that audits authoring quality and flags issues on the current page.

  1. Loads with Sidekick
  2. Reads rules from config
  3. Scans page content
  4. Assigns warnings and errors
  5. Calculates score
  6. Renders badge and issue tray

Learn more about Content Score in the README

Quality Checks

  • 🖼️

    Alt Text

    • Decorative images with redundant alt text
    • Image-only links or buttons missing alt text
  • 🔤

    Headings

    • Missing h1
    • Multiple h1s
    • Skipped heading levels
  • 🔗

    Links & Buttons

    • Generic link text (e.g. "click here", "more")
    • Same link text repeated within a section
  • 🔁

    Nested Blocks

    • Blocks nested inside other blocks
  • 📐

    Row/Column Spans

    • Merged cells in block tables
  • 📊

    Column Counts

    • Column bloat in a block table
  • 📋

    List-Like Content

    • Paragraphs starting with bullet characters
    • Paragraphs starting with numbers
  • 📦

    Block Sprawl

    • Block bloat on a page
    • Variant bloat on a page
