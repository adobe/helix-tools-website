Media Library Backfill

The Media Library Backfill tool crawls all pages on a site, parses their content for media references, and exports a historical medialog import bundle for the CLI in tools/media-library-backfill/cli.

To use this tool, you must first log in to the relevant site in your user profile on the top right in the header of this page.
