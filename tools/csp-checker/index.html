CSP Checker - Operational Telemetry Compatibility

Operational Telemetry (OpTel) collected from one of the third-party hosts used by Adobe can be blocked by websites with a restrictive Content Security Policy (CSP). To check if this is an issue, paste the site’s Content-Security-Policy header below. The tool checks whether the selected host is allowed by script-src. If script-src is omitted, default-src is used as fallback.
